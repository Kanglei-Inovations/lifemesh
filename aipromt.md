PROJECT NAME:
LifeMesh

TAGLINE:
Offline. Together.

PROJECT TYPE:
Futuristic AI-powered Offline Mesh Social Network App

PLATFORM:
Flutter (Android first, later iOS/Desktop)

DESIGN STYLE:
Ultra modern futuristic cyber UI with dark theme, neon glow, glassmorphism, animated mesh networks, AI-inspired visuals, smooth transitions, premium startup-level design.

OVERALL APP CONCEPT:
Create a futuristic mobile app called “LifeMesh” where nearby smartphones create an offline mesh network without internet using Bluetooth, WiFi Direct, and peer-to-peer communication.

The app should feel like:
- the future internet
- decentralized social network
- AI memory system
- cyber communication platform

The app combines:
- offline nearby social feed
- peer-to-peer messaging
- AI timeline memory reconstruction
- offline marketplace
- file sharing
- disaster emergency communication
- local multiplayer gaming

The UI should feel:
- cinematic
- futuristic
- alive
- premium
- highly animated
- minimal but advanced

IMPORTANT:
Do NOT create generic social media UI.
This app must look like:
“technology from the future”.
