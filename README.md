# LifeMesh

> **Offline. Together.**
> The world's first AI-powered offline mesh social network.

---

# Overview

LifeMesh is a futuristic decentralized mobile platform that enables:

* Offline communication
* Nearby social networking
* AI-powered memory timeline
* Peer-to-peer file sharing
* Local marketplace
* Emergency disaster communication
* Real-world digital interaction

without requiring:

* Internet
* Mobile data
* Central servers

LifeMesh transforms nearby smartphones into a self-organizing mesh network using:

* Bluetooth LE
* WiFi Direct
* Local Hotspot
* Peer-to-peer communication

The platform also includes an AI-powered “Reality Timeline Engine” that reconstructs real-life events using:

* GPS history
* Photos
* Nearby interactions
* Shared files
* Activity logs
* Messages
