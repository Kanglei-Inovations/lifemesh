PROJECT NAME:
LifeMesh

TAGLINE:
Offline. Together.

PROJECT TYPE:
AI-powered futuristic offline mesh social network.

====================================================
CORE IDEA
=========

Create a futuristic Flutter application called “LifeMesh”.

LifeMesh is a decentralized offline social ecosystem where nearby smartphones connect without internet using:

* Bluetooth LE
* WiFi Direct
* Local hotspot
* Peer-to-peer mesh communication

The app combines:

* offline social networking
* AI memory timeline
* mesh messaging
* file sharing
* local marketplace
* disaster communication
* nearby multiplayer gaming

The app should feel:

* futuristic
* cyberpunk
* premium
* AI-powered
* highly animated
* startup-level polished

This is NOT a normal social media app.

It should feel like:
“The future internet after centralized networks.”
